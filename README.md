# Sales & Revenue Analysis Dashboard

## Project Overview

This project presents an interactive Sales and Revenue Analysis Dashboard built using Microsoft Power BI.

## Features

- Total Sales KPI
- Total Profit KPI
- Total Units Sold
- Gross Sales Analysis
- Product-wise Sales Analysis
- Country-wise Sales Distribution
- Monthly Sales Trend
- Profit by Segment
- Interactive Country and Year Filters

## Tools Used

- Microsoft Power BI
- Financial Sample Dataset
